# OperationResult.Demo

Este es un video de la comunidad de Puebla que con ayuda de Miguel Muñoz estuvimos viendo como implementar el patron OperationResult.

Es es una implementacion, desde mi punto de vista, bastante buena y que se peude adaptar correctamente a casi cualquier necesidad. Hay muchas otras implementaciones, ya que no es un patron que se haya definido exactametne ni se le atribuye a ningun autor. Cada persona lo implementa como quiere.

Lo que se ve en el video, es solamente, una manera de hacerlo.
